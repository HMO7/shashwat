# Shashwattech Static Website

Open `index.html` in your browser to view the fully functional modern company site for Shashwattech.

- All content and navigation are single-page, no reloads, instant.
- Built with clean HTML, modern CSS, and vanilla JS for maximum speed.
- Responsive, fully featured, and includes: Home, About, Services, Products, Contact with working form and confirmation message.
- To deploy or share, simply host all files in this directory (no backend required).
